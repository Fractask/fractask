# Code of Conduct

We use the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as our code of conduct.

## TL;DR

- Be kind. Critique ideas, not people.
- Assume good faith. Most disagreement is misunderstanding.
- Disagreement is welcome; harassment, slurs, and personal attacks are not.

## Reporting

If you see behaviour that violates this code, email **cto@sunbek.com** with the details. Reports are read by a human and stay confidential.

## Scope

This applies to all project spaces — issues, pull requests, discussions, and any related communication, including private channels related to project work.
